# Projeto C++ Básico

Este workspace contém um programa C++ simples.

## Arquivos

- `main.cpp`: programa que pede nome do usuário e soma dois números.
- `.vscode/tasks.json`: tarefa do VS Code para compilar o projeto usando `g++`.

## Como usar

1. Abra este workspace no VS Code.
2. Execute a tarefa `Compilar main.cpp` ou use `Ctrl+Shift+B`.
3. No terminal, rode `./main.exe` para executar o programa.
